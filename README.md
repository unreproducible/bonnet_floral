Floral Bonnet

This is an IoT-ish style Bonnet (Hat) intended for a Raspberry Pi Zero based board.

Obviously you can use it on anything with the appropriate header.

Includes:
* BME280 - I2C
* TSL2561 - I2C
* 128x64 OLED Screen - SPI
* RGB LED - 3xGPIO/PWM
* User Button - GPIO
* Serial Port for Kernel Debugging
